# مبادر كوتشينج — ورشة "كيف تبدأ مشروعك من المنزل برأس مال صفر"

## ما الجديد في هذا الإصدار

- صفحة هبوط مخصصة **بمنتج واحد فقط** (الورشة)، مبنية بنفس الهوية البصرية (أزرق داكن + ذهبي)
- لا يوجد أي اعتماد على إنترنت خارجي للأيقونات أو الخطوط الثقيلة (فقط خط Cairo من Google Fonts، وهو سريع وموثوق)
- **واتساب**: كل الأزرار تفتح **web.whatsapp.com** مباشرة (وليس wa.me)
- **لوحة التحكم**: تتيح لك إضافة ورش/منتجات جديدة لاحقاً بنفس بنية الورشة الحالية (الاسم، الوصف، المدة، السعر، ماذا ستتعلم، مميزات إضافية)
- **صورة الكوتش والشعار**: ملفان منفصلان (`coach.jpg` و `logo.png`) مرفقان مع المشروع، يظهران لكل زوار الموقع من أي جهاز فوراً دون أي خطوة تقنية إضافية

## بيانات الدخول للوحة التحكم

- الرابط: `admin.html`
- اسم المستخدم: `admin`
- كلمة المرور: `mubadir2025`

⚠️ غيّرها فور الدخول من: الإعدادات والأمان ← تغيير كلمة المرور

## خطوات الرفع (نفس الطريقة المعتادة)

ارفع **أربعة ملفات** هذه المرة على مستودع GitHub:
- `index.html`
- `admin.html`
- `coach.jpg` (صورتك الشخصية)
- `logo.png` (شعار مبادر كوتشينج)

ثم اربط المستودع بمشروع Vercel (Import Git Repository). بعد كل تعديل: ارفع الملفات على GitHub ← Vercel ينشر تلقائياً، أو اضغط Redeploy يدوياً من تبويب Deployments.

## تحديث الصورة الشخصية أو الشعار لاحقاً

ببساطة: استبدل ملف `coach.jpg` أو `logo.png` بصورة جديدة **بنفس الاسم بالضبط**، وارفعها على GitHub بنفس طريقة رفع باقي الملفات (Add file ← Upload files ← Commit changes). ستظهر الصورة الجديدة في الموقع فوراً بعد إعادة النشر، لكل الزوار من أي جهاز.

## إضافة ورشة/منتج جديد لاحقاً

من لوحة التحكم ← الورش والمنتجات ← "+ إضافة ورشة/منتج جديد"، املأ الحقول (الاسم، السعر، المدة، ماذا سيتعلم المشارك...) واحفظ. ستظهر تلقائياً في صفحة الهبوط (على نفس المتصفح الذي أضفتها منه — هذا الجزء يبقى محلياً في لوحة التحكم حالياً).
